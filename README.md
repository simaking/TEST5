# Sikt Student-ID Lookalike (Expo + TypeScript)

Run on Android (physical device):

1. Install deps

```bash
npm install
```

2. Start the dev server and scan QR with Expo Go on your Samsung phone

```bash
npm start
```

Notes:
- Design scales from reference 395x807 via `scale` and `vScale`.
- Colors match spec. Fonts use Inter via `@expo-google-fonts/inter`.
- Main UI is in `App.tsx`.
